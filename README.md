# Automated Linux Log Parser

## Features
- Detects failed SSH login attempts
- Extracts source IP addresses
- Counts failed login attempts
- Generates brute-force attack alerts
- Creates incident reports

## Technologies
- Python
- Kali Linux
- Linux Authentication Logs
